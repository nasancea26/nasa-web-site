# NASA Website

A static website for NASA, built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html`: Main HTML file
- `css/styles.css`: Stylesheet
- `js/script.js`: JavaScript file
- `.github/copilot-instructions.md`: Instructions for Copilot

## How to Run

Open `index.html` in a web browser.

## Features

- Responsive design
- Basic navigation
- Interactive elements
